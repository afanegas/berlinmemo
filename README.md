# Berlin Ortsteile Map (Berlin Memo)

An interactive web application designed to help users learn and test their knowledge of Berlin's 12 districts (*Bezirke*) and 97 neighborhoods (*Ortsteile*).

**Live Demo:** [https://berlinmemo-taupe.vercel.app/](https://berlinmemo-taupe.vercel.app/)

## 🚀 About the Project

Berlin is a city with a rich and complex administrative structure. This project provides a visual and interactive way to explore the city's geography. Whether you are a local resident, a student, or just curious about Berlin's layout, this tool offers an engaging experience to master the map.

## 🕹️ How to Use

The application features two primary modes:

### 1. Learning Mode (*Lernen*)
- Explore the map freely.
- Click on any neighborhood (*Ortsteil*) to instantly see its name and the district (*Bezirk*) it belongs to.
- Use this mode to familiarize yourself with the locations and boundaries.

### 2. Playing Mode (*Spielen*)
- Test your knowledge with a quiz!
- The app will prompt you to find a specific neighborhood on the map.
- You have up to 3 attempts per neighborhood.
- Track your progress, time, and accuracy through the live status bar and final statistics modal.
- **Customization:** Use the settings menu to filter the game by specific regions or create a custom selection of neighborhoods to practice.

## 🛠️ Technology Stack

This project is built with a lightweight and modern tech stack for optimal performance and simplicity:

- **Core:** HTML5, Vanilla CSS3 (with Glassmorphism design), and Vanilla JavaScript (ES6+).
- **Map Engine:** [Leaflet.js](https://leafletjs.com/) – a leading open-source library for interactive maps.
- **Build Tool:** [Vite](https://vitejs.dev/) – for a fast and modern development environment.
- **Deployment:** [Vercel](https://vercel.com/).
