# Berlin Ortsteile Map (Berlin Memo)

An interactive web application designed to help users learn and test their knowledge of Berlin's 97 districhts (*Ortsteile*).

**Live Demo:** [https://berlinmemo-taupe.vercel.app/](https://berlinmemo-taupe.vercel.app/)

## 🕹️ How to Use

The application features two primary modes:

### 1. Learning Mode (*Lernen*)
- Explore the map freely.
- Toggle between **Ortsteile** and **Quartiere** in the settings menu.
- Click on any area or marker to instantly see its name.


### 2. Playing Mode (*Spielen*)
- Test your knowledge with a quiz!
- Choose between identifying **Ortsteile** (districts) or **Quartiere** (neighborhoods).
- The app will prompt you to find a specific target on the map.
- You have up to 3 attempts per target.

- **Customization:** Use the settings menu to filter the game by specific regions or create a custom selection of districts to practice.

## 🛠️ Technology Stack

This project is built with a simple tech stack for optimal performance and simplicity:

- **Core:** HTML5, Vanilla CSS3 (with Glassmorphism design), and Vanilla JavaScript (ES6+).
- **Map Engine:** [Leaflet.js](https://leafletjs.com/) – a leading open-source library for interactive maps.
- **Build Tool:** [Vite](https://vitejs.dev/) – for a fast and modern development environment.
